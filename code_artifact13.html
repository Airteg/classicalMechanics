Прискорення елемента МСС

Згідно з рівнянням Коші, елемент середовища прискорюється, якщо є дисбаланс масових сил (гравітація) або поверхневих напруг (градієнт тиску).

Секрет: Якщо $\sigma_1 = \sigma_2$, градієнт $\partial \sigma / \partial x = 0$. Об'єм стискається, але не прискорюється (стоїть на місці). Рух починається лише при різниці напруг!
